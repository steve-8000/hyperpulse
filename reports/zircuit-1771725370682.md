## 한줄 요약
Zircuit v1.132.6 introduces a number of dependency upgrades, new configuration flags, and several source‑code removals/additions that affect the node’s RPC surface and archive‑node behaviour.  The most visible changes are: 
1. Updated Go module dependencies (DataDog, libp2p, l2‑geth, zkr‑go‑common). 
2. Added the `follow_source.go` and `driver/follow_source.go` files that enable a new *Follow‑Source* mode for the rollup engine. 
3. Re‑structured the p2p layer – new flags and a more permissive gossip implementation. 
4. Removed several legacy Safe‑module contracts from the contract tree and added `SLSStray.sol`. 
5. Minor version bump in `build/version.json` and the commit hash that now points to a newer Git state.

These changes do not break existing RPC endpoints but introduce new optional parameters and a small set of new JSON‑RPC methods that expose the follow‑source feature.
