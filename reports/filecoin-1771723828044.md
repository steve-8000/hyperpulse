## 한줄 요약
Lotus v1.35.0는 Ethereum RPC 호환성 개선, FRC‑0102 서명 포장 지원, OpenTelemetry 업데이트 등 다수의 기능·버그 수정을 포함한다. 주요 변경 사항은 RPC 에러 코드 재배치와 CLI 기능 추가이며, OpenRPC 문서가 새 버전으로 업데이트되었다.

## 운영자 중요 체크
- RPC/JSON-RPC 관련 변경 단서가 감지되었습니다.
- 외부 API 엔드포인트/버전 관련 변경 단서가 감지되었습니다.
- 아카이브 노드 운영(보관/색인/동기화) 관련 변경 단서가 감지되었습니다.
- 노드 설정값/프로토콜 플래그 관련 변경 단서가 감지되었습니다.

## 주요 변경점
- EActorNotFound ↔ EExecutionReverted 에러 코드 교체
- FRC‑0102 서명 포장 지원(클라이언트 `wallet sign/verify` 명령)
- OpenRPC JSON에서 `Message` 객체 단순화(불필요 필드 제거)
- OpenTelemetry semconv v1.39.0 업데이트

## RPC/API 영향
