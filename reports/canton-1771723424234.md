## 한줄 요약
본 릴리즈는 주로 CI/CD 파이프라인과 Docker 이미지 빌드 환경을 개선하고, 일부 내부 모듈 및 테스트 파일을 정리한 버전이다. 주요 변경 사항은 빌드용 Dockerfile 추가, Postgres용 쿼리 통계 이미지 도입, 그리고 일부 메트릭스 및 비동기 코드 정리이다. RPC/JSON‑RPC 인터페이스에는 직접적인 변경이 없으며, 노드 운영에 대한 영향은 주로 빌드 이미지와 CI 설정에 한정된다.

## 운영자 중요 체크
- 외부 API 엔드포인트/버전 관련 변경 단서가 감지되었습니다.
- 아카이브 노드 운영(보관/색인/동기화) 관련 변경 단서가 감지되었습니다.
- 노드 설정값/프로토콜 플래그 관련 변경 단서가 감지되었습니다.

## 주요 변경점
- 새로운 Dockerfile.canton-build 도입으로 빌드 컨테이너에 Nix 환경을 추가
- 새로운 Dockerfile.query‑stats‑postgres 도입으로 Postgres 17 기반 통계 이미지 생성
- CI 설정 파일 대량 추가 및 수정(버전 관리, 빌드 스크립트)
- 메트릭스 모듈 일부 파일 삭제 및 리팩터링
